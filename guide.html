The Leros Canvas — 혁신적 구현 가이드
①

Zero-Layout 구조

전통적인 Header/Footer/Section 경계를 완전히 제거합니다. 모든 콘텐츠는 position: fixed로 고정되고 스크롤에 따라 변형됩니다.

/* 핵심 CSS */
.canvas-section {
  position: fixed;
  width: 100vw;
  height: 100vh;
  display: flex;
  align-items: center;
  justify-content: center;
}
②

Z-Axis Scroll (공간 확장)

스크롤 값을 CSS transform: scale()과 opacity에 매핑하여 '공간을 파고드는' 느낌을 구현합니다.

/* Scroll Observer JS */
window.addEventListener('scroll', () => {
  const scrollPercent = window.scrollY / 
    (document.body.scrollHeight - window.innerHeight);
  
  // 심볼 확장: 1 → 3배
  symbol.style.transform = `scale(${1 + scrollPercent * 2})`;
  
  // 텍스트 서서히 등장
  headline.style.opacity = Math.min(scrollPercent * 2, 1);
});
③

Underground Menu (숨겨진 푸터)

화면 하단 5vh 영역에 호버 시에만 법인 정보와 개인정보처리방침이 담긴 메뉴가 슬라이드업으로 등장합니다.

/* HTML 구조 */
<div class="underground-trigger"></div>
<div class="underground-menu">
  © 레로스 | 대표자: 윤승현
  <a href="privacy.html">개인정보처리방침</a>
</div>

/* CSS */
.underground-menu {
  bottom: -100%;
  transition: bottom 0.4s cubic-bezier(0.25, 1, 0.5, 1);
}
.underground-menu.visible {
  bottom: 0;
}
④

GPU 가속 최적화

모든 애니메이션 요소에 will-change: transform과 transform: translateZ(0)을 적용하여 60fps를 유지합니다.

.symbol, .headline, .project-icon {
  will-change: transform, opacity;
  transform: translateZ(0);
}

  • 심사관의 개인정보처리방침 접근 방해 없음
  • Zero-Layout 구현 (Header/Footer/Section 경계 제거)
  • privacy.html 페이지 존재 (404 아님)
  • 모바일 375px 정상 렌더링 확인
  • 데스크탑 1280px 정상 렌더링 확인
  • 스크롤 애니메이션 60fps 유지
